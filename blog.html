Question-1: What are the different ways to select an element in the DOM?

There are several ways to get elements from the DOM using JavaScript.These are (1) getElementById()= It selects a single element with a id. (2) getElementsByClassName()= It selects a collection of elements with any class name. (3) getElementsByTagName()=It selects a collection of elements by a tag name .(4) querySelector()=It selects the first element that matches with selector. (5) querySelectorAll()= Selects all elements that match with the selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Gets or sets the Html content of an element, including any Html tags.
innerText: Gets or sets the text content of an element.
textContent: It returns all the text content, even if it's hidden by CSS.

Question-3:What is event delegation in the DOM?

Event delegation is a process by which we can attach a single event listener to a parent element to handle events that happen on its child elements.

Question-4:What is event bubbling in the DOM?

Event bubbling is the order in which events are handled in the DOM. When an event occurs on an element, it first triggers the event handlers on that element, then on its parent, then its grandparent and all the way up to the document root.

Question-5: How do you create, add, and remove elements using JavaScript?

document.createElement("p",'div'): Creates a new element with any tag name.
parentElement.appendChild(newElement): Adds any newElement as the last child of parentElement
parentElement.removeChild(childElement): Removes any childElement from a parentElement.
